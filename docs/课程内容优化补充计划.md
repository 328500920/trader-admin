### 6.2 题库示例

#### 第1周测验：K线基础

```
1. [单选] 锤子线的特征是？
A. 上影线长，下影线短
B. 下影线长，上影线短或无
C. 上下影线都很长
D. 没有影线
答案：B
解析：锤子线的下影线长度至少是实体的2倍，上影线很短或没有，出现在下跌趋势末端是看涨信号。

2. [判断] 十字星单独出现就是强烈的反转信号。
答案：错误
解析：十字星表示多空力量均衡，单独使用意义不大，需要结合位置和后续K线判断方向。

3. [多选] 以下哪些是看跌的K线形态？
A. 射击之星
B. 锤子线
C. 黄昏之星
D. 看跌吞没
答案：A、C、D
解析：锤子线是看涨形态，其他三个都是看跌形态。
```

#### 第10周测验：风险管理

```
1. [单选] 按照2%风险规则，10000U资金单笔最大亏损应该是？
A. 100U
B. 200U
C. 500U
D. 1000U
答案：B
解析：10000U × 2% = 200U

2. [计算] 资金10000U，风险比例2%，入场价50000，止损价49000，应该开多大仓位？
答案：0.2 BTC
解析：风险金额 = 10000 × 2% = 200U
止损距离 = 50000 - 49000 = 1000U/BTC
仓位 = 200 / 1000 = 0.2 BTC

3. [判断] 亏损50%后需要盈利50%才能回本。
答案：错误
解析：亏损50%后需要盈利100%才能回本。
```

```sql

CREATE TABLE learn_case (
    id BIGINT PRIMARY KEY AUTO_INCREMENT,
    chapter_id BIGINT NOT NULL COMMENT '章节ID',
    title VARCHAR(200) NOT NULL COMMENT '案例标题',
    case_type VARCHAR(20) COMMENT '案例类型：success/failure',

    content LONGTEXT COMMENT '案例内容(Markdown)',
    sort_order INT DEFAULT 0,
    status TINYINT DEFAULT 1,
    create_time DATETIME DEFAULT CURRENT_TIMESTAMP,
    INDEX idx_chapter_id (chapter_id)
) COMMENT='实战案例表';

-- 外部资源表
CREATE TABLE learn_resource (

    platform VARCHAR(50) COMMENT '平台：youtube/bilibili/tradingview等',
    url VARCHAR(500) NOT NULL COMMENT '资源链接',
    description VARCHAR(500) COMMENT '资源描述',
    sort_order INT DEFAULT 0,
    status TINYINT DEFAULT 1,

    INDEX idx_chapter_id (chapter_id)

-- 测验题目表
CREATE TABLE learn_quiz (
```
